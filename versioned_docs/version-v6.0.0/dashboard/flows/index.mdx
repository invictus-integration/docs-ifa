# Dashboard Homepage: Overview of flow statuses
The Dashboard home page shows you an overview of the current status of all your [flows](../../support/help-center-business.mdx?q=flow). The aggregated status of the messages received by that flow can be:
* <FontAwesomeIcon icon={faCircle} color="#00a2ff" /> **Active:** messages currently processed by the flow.
* <FontAwesomeIcon icon={faCircle} color="#059669" /> **Completed:** messages completed successfully by the flow.
* <FontAwesomeIcon icon={faCircle} color="#f43f5e" /> **Error:** messages suspended, dehydrated or terminated by the flow.

<Admonition type="info" icon={<FontAwesomeIcon icon={faArrowPointer} />} title="Click on flow name or status">
Clicking on the flow name redirects you to the details of that flow, while clicking on a status redirects you to the [search page](./02_search.mdx) to list all flows of that status. 
</Admonition>

## Organizing flows in folders
To have a clear overview of your desired flows, the Invictus Dashboard allows you to organize your flows into a folder hierarchy.

The following hierarchy shows flows <Flow label="Invoices" /> and <Flow label="Total cost" /> organized in a folder named <Folder label="Invoices" />, which is under the root folder. The flow <Flow label="Inbound" /> is further organized in a sub-folder named <Folder label="Customer" />.

<FolderHierarchy items={[

    type: 'folder',
    name: 'Invoices',
    children: [
      {
        type: 'folder',
        name: 'Customer',
        children: [
          { type: 'flow', name: 'Inbound' }
        ]
      },
      { type: 'flow', name: 'Total cost' },
    ]
  }
]} />

:::tip[Construct flow detail URL with breadcrumb path]
You can construct a flow detail URL that redirects users to the details page of the flow, but following the breadcrumb path of the flow.

<FontAwesomeIcon icon={faLink} /> `https://<your-dashboard-url>/flowpath/Invoices/Customer/Inbound`

Make sure that both the folder and the flow's name is unique horizontally so that the breadcrumb only points to a single flow.
:::
