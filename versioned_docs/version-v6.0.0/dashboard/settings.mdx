## Connected Dashboard

{/* vale Microsoft.Terms = NO */}
You can start from one flow in cloud and continue on prem (and vice versa).
It provides a click-through experience between the two different Dashboards.

A hybrid (BizTalk + Azure) setup requires a link from one Dashboard (BizTalk) to the other Dashboard (Azure) for hybrid flows. 
{/* vale Microsoft.Terms = YES */}

<Walkthrough>
  <Collapsible title="Provide base URL of other Dashboard">
    Locate the <TextInput label="Connected Dashboard URL" /> input on the Settings page.
    This base URL should point to the other Dashboard which you want to connect to.

  </Collapsible>
  <Collapsible title="Enable Connected Dashboard for a flow">
    Toggle the <ToggleInput label="Connected Dashboard" /> for the desired flow in its advanced settings.

  </Collapsible>
  <Collapsible title="Query messages from that flow">
    Search for messages of this configured flow and notice a new <FontAwesomeIcon icon={faUpRightFromSquare} /> button for each message.
    This navigates you to the Connected Dashboard URL for that message chain.
  </Collapsible>
</Walkthrough>
